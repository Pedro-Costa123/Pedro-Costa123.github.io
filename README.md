## Installation
1. Clone the repo
   ```
   git clone https://github.com/Pedro-Costa123/Pedro-Costa123.github.io.git
   ```
3. Install NPM packages
   ```
   npm install
   ```
4. Run in local server
   ```
   npm start
   ```
